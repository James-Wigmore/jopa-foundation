Drop these files into this folder (exact filenames matter — the site
already references them):

  logo.png        Your logo, transparent background. Shows in the navbar
                   and footer automatically once added. Recommended:
                   ~200px tall, PNG or SVG (rename .svg files still work
                   if you update the file extension in components/navbar.html
                   and components/footer.html).

  favicon.png      Small square icon shown in browser tabs.

  og-image.png     Preview image shown when the site is shared on social
                    media / messaging apps (e.g. WhatsApp, Facebook).

Optional — replace placeholder gradient blocks with real photos:
  Any image you want to use on the About, Projects, or Gallery pages.
  Those currently show styled gradient placeholders with a caption
  ("Replace with real photo") — swap the CSS background for a real
  <img> tag once you have photos, or ask for help doing this.

Nothing in this folder is required for the site to work — every
reference has a graceful fallback (e.g. the logo falls back to a text
mark if logo.png is missing) — but adding these will make the site look
finished rather than placeholder.
